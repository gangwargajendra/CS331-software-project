# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-28

### Added
- Initial release of Smart Traffic Signal System
- YOLO-based vehicle detection
- Dynamic traffic signal timing based on vehicle count
- Emergency vehicle detection and priority handling
- Web-based frontend for monitoring
- Traffic data logging to CSV
- Configuration management system
- Utility functions for traffic analysis

### Features
- Real-time vehicle counting (cars, trucks, bikes)
- Automatic green light duration adjustment
- Emergency mode for ambulances and fire trucks
- Visual overlay with traffic statistics
- Logging system for analytics

### Technical
- YOLOv8 object detection integration
- OpenCV for video processing
- Modular architecture with config files
- Unit tests for core functionality

## [Unreleased]
